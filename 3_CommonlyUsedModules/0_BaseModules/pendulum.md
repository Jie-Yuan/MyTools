## 加减
```python
from datetime import timedelta # days
p.add_timedelta(timedelta(10))
p.subtract_timedelta(timedelta(10))
p.add() # 推荐
p.subtract() 
str(p.diff(p.add(1))) # pendulum对象作差
```
