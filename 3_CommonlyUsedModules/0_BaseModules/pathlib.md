## 判断
```
p.is_dir() # 判断p是不是目录
p.is_file() # 判断p是不是文件
p.is_absolute() # 判断p是不是绝对路径

p.match('/algor/yuanjie/Competitions/1_糖尿病/*') # 判断p是否符合某一模式
list(p.glob('*.csv')) # 在p下搜索符合某一模式文件
list(p.rglob('*.csv'))

# 利用relative_to获取当前目录\文件名 
p.relative_to(p.parent)

# 当p不是目录时，将其创建为目录
p.mkdir()
# 当p是空目录时，移除p
p.rmdir()
```
