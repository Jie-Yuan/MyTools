# 数据分析心得
- 主要目的：
    - 提出假设并验证假设，归根结底就是维度和度量（指标）的结合：做判断用对比，找原因用细分（定性与定量分析）
- 基本原则：
    - 从维度和指标组合的矩阵中选择业务场景
    - 维度和度量可以组合加工衍生出新的维度或者度量（对于一个数据项而言，它到底是维度还是度量，根据用户需求而定）
- 基本流程：
    - 确定业务目标
    - 细化业务场景
    - 转化为分析需求
    - 数据准备(数据采集/整理)
    - 数据建模
    - 模型验证
    - 数据报告/可视化

> 如果是从0开始，那最应该做的就是了解行业信息，行业内top5公司怎么做的，行业的业务需求和趋势是什么
